# Contributing to Infinity Invoice

Thanks for your interest in contributing to Infinity Invoice.

## Getting Started

1. Fork the repository and create a branch from `main`.
2. Install dependencies:
   ```bash
   npm ci
   ```
3. Run the app locally:
   ```bash
   npm run dev
   ```

## Before You Open a Pull Request

- Keep changes focused and small.
- Run checks locally:
  ```bash
  npm run lint
  npm run build
  ```
- Include screenshots for UI changes.
- Add clear testing steps in your PR description.

## Commit and PR Guidelines

- Use clear commit messages.
- Reference related issues using `Closes #<issue-number>` when applicable.
- Be respectful and constructive in code reviews.

## Need Help?

Open a discussion or an issue with context and reproduction details.
