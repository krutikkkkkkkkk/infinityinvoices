# Security Policy

## Supported Versions

Security updates are currently provided for the latest version on the `main` branch.

## Reporting a Vulnerability

Please do **not** open public GitHub issues for security vulnerabilities.

Instead, report vulnerabilities privately by emailing:

- support@infinityinvoice.app

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested remediation

We will acknowledge reports as quickly as possible and work on a fix.
